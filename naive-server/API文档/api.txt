api文档
一、server概要
该server使用了golang语言来实现，使用gorm来操作数据库，使用gin框架搭建服务器。

main包中mian.go具有一些开关和数据库.db文件，开关有:
1.log的显示等级（默认为info级）
2.注册后是否自动登录

mydatabase包中包含数据库的相关操作，以及token的简易生成

loggenerator包中包含log日志的创建和写入操作，并提供了一些方法用于写入日志到文件中

myserver包中代码包含服务器的开启、前端信息的接收和第一步处理，以及向前端返回相关信息

二、api以及错误代码
checkin：签到会返回随机1-100的分值points
错误代码：
1 token没有或无效
2 已经签到过了
3 没有登录
999 未知错误



clear：删除数据库中所有内容，需要提供一个admin_key，值为字符串114514
错误代码：444 admin_key不正确

ping：检查联通状态
暂时没有返回错误代码（如果不联通会可能会返回404？）

signin：登录
错误代码：
1 用户不存在
2 密码错误
3 已经登录
999 未知错误

signup：注册
错误代码：
1 用户名重复
2 密码为空
3 用户名为空
999 未知错误
